Task 07 👨🏻‍💻
Javascript Integration with Docker
__________________________________________

Step 1:- Start docker services
# systemctl start docker
# systemctl status docker

Step 2:- Launch container
# docker run -it --name myos -p 1234:80 centos

Step 3:- Install Python3, httpd, sudo
# yum install python3 httpd sudo -y

Step 4:- Start httpd service
# httpd

Step 5:- Come out of container
Ctrl + p +q

Step 6:- Copy html and python cgi program to container
# docker cp /var/www/html/linux.html myos:/var/www/html/
# docker cp /var/www/cgi-bin/linux.py myos:/var/www/cgi-bin/

Step 7:- Make python cgi program executable
# chmod +x /var/www/cgi-bin/linux.py

Step 8:- Give apache user sudo powers
# vi /etc/sudoers
apache ALL=(ALL) NOPASSWD: ALL
